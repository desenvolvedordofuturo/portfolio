JavaScript

Linguagem de Programação, que é muito boa e facil de aprender em comparações com outras, durante uma década está linguagem só vem crescendo ano após anos. Mas se você deveja aprender JavaScript você só precisa ter conhecimentos básicos de informática, além de compreender códigos HTML e CSS.


CSS3

Linguagem de estilização está linguagem ela da a cor deixa a sua pagina mais bonita e deixa tudo com uma página mais clem.


HTML5

Linguagem de marcação de hiertexto, que suporta tags, elementos e conteúdo multimídia — assim como marcações de documento melhoradas e novas APIs. O HTML serve, basicamente, para o desenvolvimento de páginas que contenham conteúdos simples, como textos, imagens, áudio, vídeos e etc.


Linguagem de Programação

É por onde o hardware (máquina) e o programador se comunicam. É uma linguagem formal que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas.


Python

Isso mesmo! Python é uma linguagem de programação, feita e pensada para ser fácil de aprender. É por isso que comparada a outras linguagens como C++ ou Java, o Python é muito mais facilmente compreendido até por quem não sabe nada de programação.


GitHub

GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo


Git

O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux. Um número impressionante de projetos de software depende do Git para controle de versão, incluindo projetos comerciais e de código-fonte aberto.
